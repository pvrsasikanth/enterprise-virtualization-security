# 🛡️ Enterprise Virtualization & Security (2006–2008)

### **Strategic Overview**
This repository documents the architecture and security frameworks established during my tenure at **ICICI Bank / 3i-Infotech**. It marks a pivotal era of pioneering virtualization and Linux-based infrastructure security within a high-stakes financial environment.

---

### **🏗️ Core Architectural Pillars**

* **Pioneering VMware Implementation:** Led the first-ever installation of the **VMware Virtual Centre Management Server** within ICICI Bank's infrastructure, establishing a centralized management layer for the virtual estate.

* **Linux-Based Security Hardening:** Engineered and executed strict security hardening protocols for the **Linux-based OS of VMware ESX**, ensuring compliance with rigorous banking security policies.

* **High-Availability (HA) Design:** Implemented mission-critical features including **VMotion** and **Resource Pooling** to ensure zero-downtime operations for core banking departments like Internet Banking and Credit Cards.

* **Infrastructure Modernization:** Orchestrated **Server Consolidation** projects in coordination with leading vendors (VMware, IBM, Sun Microsystems) to optimize datacenter footprints.

* **Service Resilience:** Managed and hardened **IIS 5.0/6.0** and **Windows Cluster Services**, maintaining 24/7 availability for diverse banking applications.

---

### **🚀 The "Evolution" Bridge**
My experience in **Linux Hardening** and **VMware management** at the kernel level provides the deep technical foundation required for modern **OpenShift** administration. Managing a secure Linux-based hypervisor for a major bank is directly comparable to managing secure, enterprise-grade container nodes today.
